# Project RAG

A collection of Retrieval-Augmented Generation (RAG) projects, experiments, architectures, and production-ready implementations covering modern AI application patterns.

This repository serves as a learning, research, and engineering portfolio for building scalable RAG systems using:

* Vector Databases
* Knowledge Graphs
* Hybrid Retrieval
* Agentic Workflows
* LangGraph
* FastAPI
* Evaluation Frameworks
* Production Deployment Patterns

---

## Repository Goals

The objective of this repository is to explore different RAG architectures and compare their strengths, limitations, and production readiness.

Topics covered include:

* Traditional RAG
* Hybrid Search RAG
* GraphRAG
* Agentic RAG
* Long Context RAG
* Scientific Decision Support Systems
* Evaluation and Observability
* Production Deployment
